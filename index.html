React-Flux Web Workers Demo

a simplistic flux implementation for using web workers

The Spec

We start with an array of 2134 values, and shuffle it to create 200 unique sequences.

We import an excerpt of the d3 library to do basis interpolation for each sequence.

We dynamically render an SVG for each sequence, using the interpolated line as data

The Implementations

source

SEQUENTIAL - the interpolation step will block

ASYNC - interpolation step is wrapped in a setTimeout()

WEBWORKERS - the store invokes the web worker functions using the worker API
